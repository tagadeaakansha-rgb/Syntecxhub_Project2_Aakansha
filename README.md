#  Online Shopping Cart System (Java)

This project is a console-based shopping cart system developed using **Java**, as part of my internship at **Syntecxhub**.

##  Features
- Add products to cart
- View cart items
- Update product quantity
- Generate final bill
- Uses ArrayList and HashMap

## Concepts Used
- Object-Oriented Programming
- Java Collections Framework
- User input handling with Scanner

## Technologies
- Java
- IntelliJ IDEA

##  How to Run
1. Open the project in any Java IDE
2. Run `ShoppingCart.java`
3. Follow the on-screen menu
